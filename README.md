
# Monitor de Paginas

Sistema de monitoreo que verifica cada 10 minutos si estan en linea las paginas y servicios que administro, se guardan las consultas con el tiempo de respuestas y se genera un grafico del historial, al retornar un status diferente a 200 me envia un mensaje en Telegram con el bot configurado y/o al email.
